初始化本地库：git init
配置仓库级别：git config
配置系统用户级别：git config --global

配置签名（保存在.git/config文件中）：
git config user.name tom_pro
git config user.email goodMorning_pro@atguigu.com
查看状态：git status
添加到暂存区：git add good.txt
从暂存区移除：git rm --cache good.txt
提交到本地库：git commit -m "注释" good.txt
查看历史记录：git log --pretty=oneline
查看历史记录：git reflog
回到某个版本：git reset --hard/--soft/--mixed 01ca49e

分支：
查看分支：git branch -v
创建一个hot_fix分支：git branch hot_fix
切换到hot_fix分支：git checkout hot_fix
将hot_fix分支合并到当前分支：git merge hot_fix
如果有冲突，解决冲突后，git commit时不能带文件名
远程：
添加远程地址别名：git remote add 别名 远程地址
查看远程地址：git remote -v
推送：git push 远程地址别名 分支
克隆：git clone 远程地址
拉取：git fetch 远程库地址别名 远程分支名
合并：git merge 远程库地址别名/远程分支名
拉取并合并：git pull 远程库地址别名 远程分支名
